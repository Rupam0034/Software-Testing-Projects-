# Test Plan – Amazon Testing Project

## 1. Objective

To verify the functionality of Login and Add to Cart features of the application and ensure they work as expected.

---

## 2. Scope

**In Scope:**

* Login functionality
* Add to Cart functionality
* Product search

**Out of Scope:**

* Payment process
* Performance testing

---

## 3. Testing Approach

* Test cases are designed for all scenarios
* Test cases are executed manually
* Critical scenarios are automated using Selenium IDE
* Bugs are reported using Mantis

---

## 4. Test Environment

* Browser: Google Chrome
* OS: Windows
* Tools: Selenium IDE, Mantis

---

## 5. Test Scenarios

* Login with valid credentials
* Login with invalid credentials
* Login with empty fields
* Search product
* Add product to cart

---

## 6. Entry Criteria

* Application is accessible
* Test data is available

---

## 7. Exit Criteria

* All test cases executed
* Critical defects are resolved

---

## 8. Deliverables

* Test Plan
* Test Cases
* Bug Reports
* Automation File (.side)
